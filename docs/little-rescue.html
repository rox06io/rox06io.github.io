Little Rescue

Mobile app


Description

This game for mobile devices powered by Android is a 2D platform game.

The game was created for a private individual as a birthday present for his son. In it he specifies that he wants to use zombies to kill humans. As a result of working through the design and agreeing with the client, we decided to develop a platform game where you have to save your friends from zombies.

Finally, in the game, you will need to put yourself in the protagonist's shoes to rescue the children from their cages while killing zombies.

The game was created in three days
